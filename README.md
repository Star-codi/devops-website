## 🤝 Want to Contribute?

Contributions are welcome! Whether it's fixing a typo, improving an explanation, or adding a new module — every bit helps the community.

👉 Please read [CONTRIBUTING.md](./CONTRIBUTING.md) before submitting anything.

**The short version:**
1. Open an Issue first to propose your change
2. Wait for approval
3. Fork the repo and make your changes in Markdown
4. Submit a Pull Request

---

## 🛡️ Branch Protection

The `main` branch is protected. No one can push directly to it — including contributors. All changes go through a Pull Request and are reviewed before merging.
